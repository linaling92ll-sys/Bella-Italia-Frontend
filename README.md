# Bella-Italia Frontend
Bella-Italia är en responsiv webbplats för en italiensk restaurang. Webbplatsen är utvecklad med HTML, CSS och JavaScript.

## Funktioner

### Startsida
- Presentation av restaurangen
- Navigation till övriga sidor

### Menysida
- Hämtar maträtter från ett API
- Visar maträtter uppdelade i kategorier:
  - Pizza
  - Pasta
  - Efterrätter
  - Drycker

### Kontaktsida
- Visar restaurangens kontaktuppgifter

### Adminpanel
- Inloggning för administratör
- Lägg till nya maträtter
- Redigera befintliga maträtter
- Ta bort maträtter
- Logga ut

## Tekniker

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API
- Local Storage

## Tillgänglighet

Webbplatsen har testats med WAVE för att identifiera eventuella tillgänglighetsproblem.

## Responsiv design

Webbplatsen är anpassad för olika skärmstorlekar med hjälp av CSS.
